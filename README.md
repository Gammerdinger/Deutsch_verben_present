# Deutsch_verben_present
An app to quiz German present tense conjugation.

## Patch notes

Patch 1.1

- Added ä, ö, ü and ß buttons for users with English keyboards to more easier use those letters
- Added a conjugation table once you’ve checked your answer (will also help us spot errors in the database easier)
- Added functionality to sort the verbs by lots of categories (reflexive, separable, irregular (and also specifically reasons for being irregular like a to ä stem-changers))
- Added functionality to sort by which pronouns you want to test
- Added definitions underneath the infinitives
